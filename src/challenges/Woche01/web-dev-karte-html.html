Woche 01: Web Dev Karte in HTML & CSS

Beschreibung

Diese Woche dreht sich alles um HTML und Strukturierung. Deine Aufgabe ist es, eine eigene Spielkarte im Stil von Pokémon oder Quartett zu gestalten – aber mit dem Thema Webentwicklung.

Ziel ist es, mit reinem HTML eine sinnvolle Struktur aufzubauen. Kein CSS, kein JavaScript!
Abgabe ist am Freitag, den 7. November 2025.

Ziel

Erstelle eine Spielkarte, die optisch und inhaltlich an eine Sammelkarte erinnert. Verwende dabei nur HTML-Tags und Klassen, um deine Karte später leicht gestalten zu können.

Anforderungen

  • Die Karte soll folgende Bereiche enthalten:
    • Name / Titel der Figur
    • Rolle oder kurze Beschreibung
    • Bild (z. B. Platzhalter oder eigenes Motiv)
    • Eigenschaften oder Fähigkeiten (frei wählbar)
    • Werte oder Symbole (z. B. ☕ = Kaffee, HP-Punkte usw.)
  • Verwende semantische HTML-Tags und passende Klassennamen (z. B. .card, .card-body, .ability, .stats).
  • Der Inhalt darf frei erfunden und kreativ gestaltet werden.
  • Es soll noch kein CSS oder Styling verwendet werden – Fokus liegt auf der HTML-Struktur.
