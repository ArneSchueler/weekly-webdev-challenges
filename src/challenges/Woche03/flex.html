Woche 03: Responsive Design (Desktop / Tablet / Mobile Cards)

Beschreibung

Diese Woche dreht sich alles um Flexbox, Grid und Responsive Design. Du lernst, wie du Layouts baust, die sich je nach Bildschirmgröße automatisch anpassen — von Desktop über Tablet bis Mobile.
NDeine Aufgabe ist es, drei thematische Karten zu erstellen, die unterschiedliche Gerätegrößen repräsentieren: Desktop, Tablet und Mobile. Jede Karte soll eigene Inhalte haben und sich visuell von den anderen unterscheiden.

Das Besondere: Das Layout der Karten ändert sich abhängig von der Bildschirmbreite:

  • Desktop: alle 3 Karten nebeneinander
  • Tablet: die Karten „wrappen“ – 2 oben, 1 unten
  • Mobile: alle Karten untereinander

Je nach Bildschirmbreite wird außerdem eine andere Karte hervorgehoben.
Abgabe: am Freitag, 21. November 2025.

Ziel

Nutze die Layout-Techniken aus CSS, um:

  • Flexbox und Grid zur Strukturierung deiner Seite einzusetzen
  • ein responsives Layout zu erstellen, das sich an unterschiedliche Bildschirmgrößen anpasst
  • Inhalte übersichtlich und modern zu präsentieren
  • einzelne Elemente (Cards) je nach Viewport gezielt hervorzuheben

Anforderungen

  • Verwende eine separate CSS-Datei (z. B. style.css)
  • Erstelle drei Karten, die folgende Themen darstellen:
    • Desktop Layout
    • Tablet Layout
    • Mobile Layout
  • Die Cards sollen über Flexbox oder Grid angeordnet werden.
  • Jede Karte enthält:
    • ein Titelbild
    • ein Icon
    • einen Titel
    • einen kurzen Beschreibungstext
    • einen Button
  • Setze mindestens drei Media Queries ein:
    • Desktop (≥1024px) → Card Desktop hervorheben
    • Tablet (600px–1023px) → Card Tablet hervorheben
    • Mobile ( kleiner als 600px) → Card Mobile hervorheben
  • Die Hervorhebung kann erfolgen durch:
    • größere Darstellung/li>
    • stärkeren Schatten/li>
    • Glow
    • anderen Hintergrund oder Gradient
  • Verwende keine absolute Positionierung und kein JavaScript.
  • Arbeite klar strukturiert und übersichtlich, orientiere dich gern an modernen UI-Cards.
